Project Overview

This is a simple microprocessor-based game where the player dodges falling bricks. Bricks fall from the top of the screen in pairs, and the player must maneuver to avoid being hit. The game increases in difficulty as brick speed increases.

Gameplay & Controls

Movement: Use the buttons (or optional keypad) to move left and right.

Restart: A dedicated button allows restarting the game at any time.

Bricks: Three types of bricks appear randomly, each falling faster than the last.

Score & Rounds: Avoid bricks to complete rounds. Score increases with each successful round.

LEDs & Sounds

LEDs:

Green flashes on successful round completion.

Red flashes on collision (game over).

Sound:

Optional buzzer sounds: “check” on passing a round, “loss” on hit.

Players can toggle sound on or off.

Optional Features

High Score: Tracks the player’s highest score (if time allows).

Serial Port: Debug outputs and optional sprite control using keypad.

How to Play

Start the game from the main menu.

Move left or right to avoid falling bricks.

Survive as many rounds as possible.

Restart with the dedicated button after a collision.
